## PDF Tools 

This tool is for pdf related works 

### Modules used ? 
- PyPDF2
- OS 

### What can it do ? 
- Merge PDFs
- Split PDFs 
- Add Watermark 
- Rotate PDF 
- Encript PDF
- Reoder PDF 

### Setup Instructions
- pip3 install -r requirements.txt
- Add the path of pdfs and you are good to go ;)
